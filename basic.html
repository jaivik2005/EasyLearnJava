What is Java?

📌 Java is a high-level, object-oriented, platform-independent, general-purpose programming language developed by James Gosling at Sun Microsystems in 1995 (now owned by Oracle).

📝 Key Features:

👉 Simple: Easy to learn and write.
👉 Object-Oriented: Uses classes and objects.
👉 Platform-Independent: “Write Once, Run Anywhere” (WORA) because Java programs run on JVM (Java Virtual Machine).
👉 Secure: No direct access to memory.
👉 Robust: Handles errors and exceptions well.
👉 Multithreaded: Can perform multiple tasks simultaneously.
