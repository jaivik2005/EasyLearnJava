✅ Java Installation Steps (Platform-wise)

💻 1. For Windows

  • 🟩 Step 1: Download JDK
    • 🔗 Visit: Oracle JDK Download
    • ➡ Choose JDK 21 or 17 (LTS)
    • ➡ Select Windows x64 Installer (e.g., jdk-21_windows-x64_bin.exe)
  • 🟩 Step 2: Install JDK
    • ✔ Run .exe file
    • ✔ Click Next → Next → Install
    • ✔ Default path: C:\Program Files\Java\jdk-21
  • 🟩 Step 3: Set Environment Variables
    • 🔍 Open Start → Search "Environment Variables" → Click Edit system environment variables
    • ✅ Add JAVA_HOME
      • → System Variables → New
      • Name: JAVA_HOME
      • Value: C:\Program Files\Java\jdk-21
    • ✅ Add to PATH
      • → System Variables → Edit Path → New:
      • %JAVA_HOME%\bin
  • 🟩 Step 4: Verify Installation
    • Open Command Prompt and run:
    java -version
javac -version
    • ✅ You should see the installed version.

🐧 2. For Ubuntu/Linux

  • 🟦 Step 1: Install JDK via APT 
    sudo apt update
sudo apt install openjdk-17-jdk
# Or use openjdk-21-jdk
  • 🟦 Step 2: Set JAVA_HOME (Optional) 
    sudo nano /etc/environment
    🔽 Add at end:
    JAVA_HOME="/usr/lib/jvm/java-17-openjdk-amd64"
    📤 Then run:
    source /etc/environment
  • 🟦 Step 3: Verify 
    java -version
javac -version
    ✅ Version should appear.

🍎 3. For macOS

  • 🟥 Step 1: Install via Homebrew 
    brew install openjdk@21
  • 🟥 Step 2: Link JDK 
    sudo ln -sfn /opt/homebrew/opt/openjdk@21/libexec/openjdk.jdk /Library/Java/JavaVirtualMachines/openjdk-21.jdk
  • 🟥 Step 3: Add to Path
    Open .zshrc or .bash_profile, add:
    export JAVA_HOME=$(/usr/libexec/java_home -v 21)
export PATH=$JAVA_HOME/bin:$PATH
    Then:
    source ~/.zshrc    # or source ~/.bash_profile
  • 🟥 Step 4: Verify 
    java -version
javac -version
    ✅ Done!

🔧 What You Need After Setup

🎯 Use a Code Editor:
  • VS Code
  • IntelliJ IDEA
  • Eclipse

▶️ Try Your First Java Program

Create a file Hello.java with this code:
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}
🛠 Compile:
javac Hello.java
🚀 Run:
java Hello
🎉 Output:
Hello, Java!
