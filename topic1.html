üîπ 2. Non-Primitive Data Types

These types are created by the programmer and can store multiple values.

  • String (sequence of characters) 
    String name = "Alice";
System.out.println(name); // Output: Alice
  • Array (collection of elements) 
    int[] numbers = {1, 2, 3, 4, 5};
System.out.println(numbers[0]); // Output: 1
  • Class (blueprint for objects) 
    class Student {
    String name;
    int age;
}

public class Main {
    public static void main(String[] args) {
        Student s1 = new Student();
        s1.name = "Bob";
        s1.age = 20;

        System.out.println(s1.name); // Output: Bob
        System.out.println(s1.age);  // Output: 20
    }
}

  • Narrowing Casting (manual) ‚ö†Ô∏è
